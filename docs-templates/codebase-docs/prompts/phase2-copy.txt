## กฎสร้างไฟล์

1. ทุกหน้าใช้ sidebar **ชุดเดียวกัน** ตาม outline — `class="active"` ที่ลิงก์หน้านั้น
2. หน้า root: `href="styles.css"` — หน้า features/: `href="../styles.css"` และลิงก์ `../` ไป root
3. แทนที่ `{{PLACEHOLDER}}` ใน template ด้วยชื่อแอป/โปรเจกต์จริง
4. เนื้อหาต้องมาจากโค้ดจริง — ไม่เดา
5. วันที่ใน `last-updated` ใช้วันนี้
6. เขียน `AI-GUIDE.md` ใหม่ให้เป็นจุดเข้าหลักสำหรับ Agent (แผนที่ไปยังหน้า HTML, path โค้ด, `.cursor/rules` / `skills` — ไม่ใส่ขั้นตอน setup)
7. อัปเดต `.cursor/.cursorrules` และ `.cursor/rules/architecture.mdc` จากโค้ดจริงของโปรเจกต์นี้ (แทน placeholder จากแพ็กติดตั้ง — ห้ามคัดลอกจากโปรเจกต์อื่น)
